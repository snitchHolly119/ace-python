# Сохранение и изменение данных
Для сохранения данных в памяти компьютера чаще всего используются переменные. Значение переменной можно изменить в любое время после её создания.

> [!NOTE]
> **Переменная** — это именованная область памяти компьютера, которая служит для хранения данных (чисел, текста, логических значений), значения которых могут изменяться в ходе выполнения программы


## Создание новых переменных
```python
name = "Harry"
age = 10
```
> [!CAUTION]
> В языке программирования Python нельзя создавать пустые переменные.

> [!CAUTION]
> Имена переменных не могут начинаться с цифры, содержать пробелы и [, +, -, {}, *. /, %, :, ;, ].


## Изменение ранее сохранённых данных в переменой
```python
age = 11
```
## Сохранение в переменную данных введённых пользователем с клавиатуры 
```python
country = input("Из какой вы страны? ")
city = input()
```
> [!NOTE]
> Функция **input** из стандартной библиотеки языка программирования Python позволяет вводить данные с клавиатуры и сохранять их в переменные для последующего использования по мере выполнения выполнения программы.

> [!IMPORTANT]
> Функция — это именованный блок кода, который выполняет конкретную задачу и может быть многократно вызван из разных частей программы.


## вывод данных на экран
```python
print(name, "возраст", age)  # вывод  данных через пробел
print(f"{name} {age} лет.")
```


# Примитивные Типы Данных
В языке программирования Python существует 4 основных примитивных типа данных:
- Целые числа (-7, 0, 12)
- Реальные числа (-0.05, 3.5, 1.75)
- Строки ("a", "Кому на Руси жить хорошо.")
- Логические значения (True, False)

[!NOTE]
> **Примитивные типы данных** — это базовые, встроенные в язык программирования типы данных, представляющие собой простейшие значения (числа, текст, логические значения. 
  

## Целые Числа (int)
```python
a = 10
```
> [!IMPORTANT]
> **Целые числа** в языке программирования Python могут содержать только знаки от 0 до 9.

## Реальные Числа (float)
```python
p = 3.14
```
> [!NOTE]
> **Вещественные числа** — это числа, у которых есть дробная часть. Они могут быть положительными или отрицательными.

> [!IMPORTANT]
> **Вещественные числа** в языке программирования Python могут содержать только знаки от 0 до 9 и точку.

## Строки (str)
Строки (текст) в языке программирования Python могут содержать любые знаки которые могут быть найдены на стандартной клавиатуре. Для обозначения начала и конца строки используется двойные или одинарные кавычки.

```python
name = "Гарри"
surname = 'Поттер'
```

## Логические Значения (bool)
Логические выражения в языке Python могут принимать одно из двух значений ложь (False) или правда (True).

```python
b = True	# Правда
c = False	# Ложь
```
## Изменение Одного Типа Данных на Другой
Для того чтобы изменить тип данных значения необходимо использовать подходящую для этих целей функцию:
- int() -  принимает реальное число или строку (текст) (состоящую из символов от 0 до 9)
- str() - принимает реальные или целое число
- float() - принимает целое число или строку (текст) (состоящую из символов от 0 до 9 и точки).

```python
n = int("20")	# 20
m = int(4.3)	# 4

h = float("1.5")	# 1.5
k = float(6)	# 6.0

s = str(125)	# "125"
t = str(10.24)	# "10.24"
```
